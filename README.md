# Bangla-Soft | Modern Software Solutions

A premium, modern landing page for Bangla-Soft, a leading software development agency in Dhaka, Bangladesh.

## Features
- **Modern Aesthetic**: Dark theme with indigo and teal accents.
- **Glassmorphism**: UI elements using backdrop filters for a professional look.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Micro-Animations**: Scroll-triggered reveal animations.
- **High-Quality Imagery**: Custom generated high-resolution assets.
- **Performance**: Lightweight HTML/CSS structure with optimized icons via Lucide.

## Tech Stack
- HTML5
- CSS3 (Custom Design System)
- Lucide Icons
- Google Fonts (Outfit)
